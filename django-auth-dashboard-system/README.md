# Django Authentication Dashboard System

A professional Django mini-project featuring a complete authentication system with user signup, login, logout, and a protected dashboard.

## Features

- User Registration (Signup)
- User Login
- User Logout
- Protected Dashboard
- Session Management
- Modern Dark/Light UI

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run migrations:
```bash
python manage.py migrate
```

3. Start the development server:
```bash
python manage.py runserver
```

4. Visit http://127.0.0.1:8000/

## Technologies

- Django 5.0.1
- Bootstrap 5
- SQLite Database
