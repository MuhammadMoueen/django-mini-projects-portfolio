# Django Authentication Dashboard System

A professional Django mini-project featuring a complete authentication system with user signup, login, logout, and a protected dashboard with modern UI.

## Features

- **User Registration** - Complete signup form with validation
- **User Login** - Secure authentication system
- **User Logout** - Session management
- **Protected Dashboard** - Login-required views
- **Profile Management** - User profile page
- **Session Management** - Django built-in sessions
- **Modern Dark/Light UI** - Theme toggle functionality
- **Responsive Design** - Mobile-friendly interface
- **Professional Metrics** - Dashboard analytics cards

## Technologies Used

- Django 5.0.1
- Python 3.x
- Bootstrap 5.3.2
- Bootstrap Icons
- SQLite Database
- HTML5/CSS3/JavaScript

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run migrations:
```bash
python manage.py migrate
```

3. Start the development server:
```bash
python manage.py runserver
```

4. Visit http://127.0.0.1:8000/
